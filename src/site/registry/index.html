A repair is never a blank cheque.

Patch the Web refuses arbitrary JavaScript and validates a patch again on your device before it can run.

  1. 01
    Scoped

    It can run only on the exact domain and path declared in the receipt.

  2. 02
    Constrained

    Only allowlisted transformations: filters, layout, accessibility, keyboard support, and local progress.

  3. 03
    Monitored

    Selector drift is detected by Compatibility Sentinel and affected patches are quarantined.
